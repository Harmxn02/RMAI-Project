\subsection{The Feedback Loop}

\textit{(the core of the section \textemdash{} a paragraph + possibly a small diagram)}

Walk through the process step by step: initial generation (using a random prompt from a predefined set) $\rightarrow$ validation $\rightarrow$ if errors are found, construct a regeneration prompt containing the incorrect code and the validation errors $\rightarrow$ regenerate $\rightarrow$ re-validate $\rightarrow$ repeat until either no errors remain or the iteration limit is reached. When the script ends you see a table of the results, showing the ERRORS, WARNINGS and INFOS counts before and after the guardrail process, with a delta (difference), and the number of iterations taken.
