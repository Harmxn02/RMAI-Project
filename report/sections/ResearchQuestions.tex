\section{Research questions}
\label{sec:research-questions}

Acknowledging the issues outlined in the~\nameref{sec:background} section, this research will address the following research questions:

\begin{description}
	\item[RQ1.] To what extent can an iterative validation-and-regeneration guardrail improve the semantic correctness of LLM-generated HTML code and ensure adherence to best practices?
	\item[RQ2.] Can the guardrail enable smaller, cheaper LLMs to achieve output quality comparable to large, cloud-based models?
\end{description}
